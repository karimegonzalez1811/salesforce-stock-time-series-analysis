# Salesforce Stock Time-Series Analysis

## Overview
Analyzed 4,385 Salesforce stock records from 2004–2021 using R, dplyr, and ggplot2.

## Tools
R, dplyr, ggplot2, PowerPoint

## Analysis
- Converted date fields
- Calculated descriptive statistics
- Calculated daily percentage changes
- Created financial data visualizations

## Visualizations
- Open/close price trends
- Yearly closing-price box plots
- Closing-price distribution
- Daily percentage-change charts
